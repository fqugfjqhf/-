所需要的库
tensorflow ==2.13.0
keras ==2.13.1
matplotlib ==3.7.2

数据集
本项目使用的数据集名为"cats_and_dogs_filtered"，包含了猫和狗的图像。数据集被分为两个主要目录："train"和"validation"，分别用于训练和验证图像。

代码功能
脚本加载预训练的InceptionV3模型，并去掉其顶部的分类层，因为它们与新的分类任务无关。
将预训练模型的所有层设置为不可训练，以防止在训练过程中更新它们的权重。
在预训练模型的顶部添加自定义分类层，以适应猫和狗分类任务。
使用RMSprop优化器和二元交叉熵损失函数对模型进行编译。
对训练集进行数据增强，以增加训练样本的多样性并减少过拟合。
设置数据生成器，以处理批量加载和处理训练和验证图像。
对模型进行训练，并在验证集上进行验证，训练一定数量的epochs。
收集每个epoch的训练和验证准确性和损失值，并使用Matplotlib绘制。
